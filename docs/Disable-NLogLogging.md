# Disable-NLogLogging

## SYNOPSIS
Disables all logging.

## SYNTAX

```
Disable-NLogLogging [<CommonParameters>]
```

## DESCRIPTION
The Disable-NLogLogging Cmdlet disables all global logging.

## EXAMPLES

### EXAMPLE 1
```
Disable-NLogLogging
```

Disable all logging
